# Website Dashboard Mutu Pelayanan Rumah Sakit Jawa Timur

This is a starter project built with React 18, Vite and Tailwind CSS. It includes:

- Landing page with hero and responsive navbar
- Akreditasi charts (Pie + horizontal Bar) using Chart.js
- Interactive indikator table with search, filter and sort
- Download cards and document manager
- Admin dashboard with login, live chart editor and data manager
- Local storage-backed state (Context API)

Quick start:

1. Install dependencies

```bash
cd /home/aniipid/web-mutu
npm install
```

2. Run dev server

```bash
npm run dev
```

Admin login:
- Username: any
- Password: admin

Notes:
- This project uses localStorage for demo persistence and a simple password check for authentication. For production use, integrate a secure backend.
